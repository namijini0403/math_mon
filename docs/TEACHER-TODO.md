  - 프로젝트 #1 = 운영(게임 저장) / 프로젝트 #2 = 분석(가명 로그) — 이름 헷갈리지 않게!

- [ ] **Railway 환경변수** 설정 (`.env.example` 참조):
  - `DATABASE_URL` = 프로젝트 #1 연결 문자열 (Session pooler URI)
  - `ANALYTICS_DB_URL` = 프로젝트 #2 연결 문자열 ← **①이 끝나기 전엔 비워두세요 (비어 있으면 수집 안 됨 = 안전장치)**

  - `TEACHER_KEY` = 교사 API 키

## ③ 구매 (예산 ≈400만, 전부 신품 조달 가능)

| 품목 | 요청 스펙 (견적서에 이대로) | 예상가 |
|---|---|---|
| Mac mini | **M4 Pro, 통합메모리 64GB**, SSD 1TB 이상 (Apple 교육기관 구매) | ≈ 250만 |
| NAS | 2베이 (Synology DS224+ 급) + **NAS용 HDD 8TB × 2** (RAID1 구성) | ≈ 90~110만 |
| UPS | 1000VA / 600W 이상, 자동 종료 연동 지원 | ≈ 30만 |
| 잡비 | CAT6 케이블, 멀티탭(서지보호) | ≈ 15만 |

- 구매 시점: **독서·감사일기 앱이 가동된 뒤**여도 늦지 않습니다 (1차 규칙 마스킹은 장비 없이 동작). 예산 소멸 시한이 있으면 먼저 사두어도 무방 — 사두면 백업 용도로 즉시 활용.
- 사지 말 것: GPU 워크스테이션(중고 3090 불가·신품 4090 예산 초과·16GB급은 금방 한계), 브랜드 워크스테이션 완제품, RTX 5090.

## ④ 학교망 협조 (정보부장·교육청 네트워크팀에 전달할 문구)

> "교내 설치 장비는 **외부에서 학교로 들어오는 연결(인바운드)을 일절 받지 않습니다.**
> 장비가 학교에서 외부로 나가는 연결(아웃바운드 HTTPS 443)만 사용하며,
> 학생 서비스는 외부 클라우드에서 동작하므로 학교망 장애·정전과 무관하게 운영됩니다.
> 요청 사항: 해당 장비의 아웃바운드 443 허용 + 고정 내부 IP 1개."
