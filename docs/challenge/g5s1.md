# 5-1 심화 유형 구현 명세 (최고수준수학 2·3단계 유형화 — 원문 미사용, 구조만)

공통: 모든 스킬 `challenge: true`, `difficulty: 3`. 답이 자연수면 fill-blanks, 분수면 fraction-input(기약).
스킬 id 규칙: `ch51-{n}`. unitId는 각 단원 id.

## unitMix (자연수의 혼합 계산)
1. `ch51-promise` 약속 연산: "가★나 = 가×나−가" 식을 정의하고 (a★b)★c 계산. 연산 정의 풀 4종(가×나−가 / 가×(가−나) / 가+나×2 / (가+나)×나 등), 중간·최종 결과 자연수·양수 보장(파라미터 역산). 해설: 안쪽부터 대입.
2. `ch51-op-place` 기호 배치 최대값: "a □ b □ c □ d에 +, −, ×를 한 번씩 넣어 가장 큰 값" — 생성기에서 3!=6가지 전부 계산(×÷ 우선순위 적용)해 최대값이 정답. 모든 배치가 자연수가 되는 수만 출제. fill-blanks.
3. `ch51-ineq` 부등식 최대 자연수: "(A−B)÷C×D > E×□" 좌변이 자연수가 되게 구성, □ 최대값 = ceil(좌변/E)−1 ≥ 1 보장.

## unitDiv (약수와 배수)
1. `ch51-gcdlcm` 역산: "두 수의 최대공약수가 G, 최소공배수가 L일 때 두 수를 구하세요(작은 수부터)" — G·서로소쌍(p,q, p<q) 선택해 L=G·p·q 구성, 서로소쌍이 유일한 (p,q)만 출제(p·q 분해가 유일하도록 p,q 모두 소수). fill-blanks 2칸.
2. `ch51-remainder` 나머지 공통: "A로 나누면 A−k, B로 나누면 B−k가 남는 수 중 N에 가장 가까운 수" (수+k가 lcm(A,B)의 배수). k=1~3, 답 양수 보장.
3. `ch51-gear` 톱니바퀴: "톱니 a개·b개 두 톱니바퀴가 처음 위치로 돌아올 때까지 작은 바퀴는 몇 바퀴?" = lcm(a,b)÷min(a,b).

## unitPattern (규칙과 대응)
1. `ch51-compose` 이중 대응: "○=□+a, ◎=○×b일 때 □=x이면 ◎는?" 또는 역방향(◎ 주고 □). 정수 보장.
2. `ch51-age` 나이 N배: "아빠 A세, 아이 B세. 아빠 나이가 아이의 n배가 되는 건 몇 년 후?" — x=(A−nB)/(n−1)이 양의 정수가 되는 조합만 출제.
3. `ch51-arith` 등차수열: "f, f+d, f+2d, …에서 처음으로 X보다 커지는 수는 몇 번째?" 답 = 번째(자연수).

## unit1 (약분과 통분)
1. `ch51-frac-range` 기약분수 개수: "기약분수로 나타내면 a/b가 되는 분수 중 분모가 P 초과 Q 미만인 것은 몇 개?" — bk 범위에서 k 개수, 2~6개가 되게 P,Q 구성.
2. `ch51-frac-add-same` 같은 수 더하기: "a/b의 분모·분자에 같은 수를 더해 c/d와 같게 — 더한 수는?" □=(ad−bc)/(c−d) 양의 정수 보장 조합만.
3. `ch51-frac-ineq` 부등식: "a/b < □/N < c/d, □에 들어갈 자연수는 모두 몇 개?" 통분 가능한 N(분모 lcm ≤ 60), 개수 2~8 보장.

## unit2 (분수의 덧셈과 뺄셈)
1. `ch51-tape` 테이프 잇기: "길이 (대분수) m 테이프 n장을 e/f m씩 겹치게 — 전체 길이는?" = n·L − (n−1)·overlap, 분모 ≤ 60 보장, fraction-input(mixed).
2. `ch51-sumdiff` 합·차: "㉮+㉯=S, ㉮−㉯=D일 때 ㉮는?" ㉮=(S+D)/2 — S,D 분모 작게(합산 분모 ≤ 60), ㉮ 기약 진분수/대분수 보장. fraction-input.
3. `ch51-telescope` 단위분수 연속합: "1/(1×2)+1/(2×3)+…+1/(n(n+1))" = n/(n+1). n=4~9. 해설에 망원 소거 원리. fraction-input.

## unitPoly (다각형의 둘레와 넓이)
1. `ch51-stairs` 계단 도형 둘레: "한 변 a cm 정사각형 n개를 계단 모양으로 쌓은 도형의 둘레" = 4·n·a (둘레는 외접 직사각형과 같음 — 해설에 이유). fill-blanks.
2. `ch51-overlap` 겹친 넓이: "직사각형(w×h)과 정사각형(s²)이 겹쳐 있고 겹친 부분이 정사각형의 1/k일 때 전체 도형의 넓이" = wh + s² − s²/k (정수 보장: k | s²).
3. `ch51-peri-eq` 둘레 같은 정다각형: "정m각형과 정n각형의 둘레가 똑같이 P cm — 한 변 길이의 차는?" P는 lcm(m,n)의 배수.
4. `ch51-rhombus-scale` 마름모 배율: "두 대각선을 각각 k배로 늘리면 넓이는 몇 배?" = k². 해설에 (ka×kb÷2)=k²(ab÷2).
