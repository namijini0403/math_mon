## ① 표기 가드가 '구조적으로' 못 잡는 것 — 텍스트 수동 점검

가드(game/curriculumGuard.ts)는 **기호 표기**만 봅니다(×·÷·분수·소수·거듭제곱·√). 다음은 가드 밖이라 사람이 읽어야 합니다.

- [ ] **1·2학년 풀이의 '말로 된' 연산 누출**: 가드는 `÷` 기호만 잡습니다. "2로 나누면", "3씩 묶으면"처럼 **나눗셈·곱셈을 말로 쓴 것**이 도입 학년보다 앞서 나오는지는 못 잡습니다. 1·2학년 풀이를 읽을 때 이 점만 눈으로 확인. (현재 1·2학년 전 단원 audit 완료 상태이나, 새 시드가 무한이라 표본 점검 권장.)
- [ ] **부등호(<, >)의 '말' 처리**: 앱 정책 = 부등호 입력은 클릭 선택, 풀이는 '말 + 기호'. 1·2학년은 기호 대신 '말'만(도입 학년 불명). 단원별로 자연스러운지.

## ② 심화(challenge) 단원 — 답안키 관례·난이도 판단

5학년 심화 풀이를 제가 audit하며 수학 오류·표기 위반은 고쳤습니다(아래 ④ 참조). 다음은 **관례·수준 판단**이라 선생님 결정이 필요합니다.

- [ ] **ch51-gcdlcm (최대공약수·최소공배수로 두 수 구하기)**: L÷G가 서로 다른 두 소수 p·q의 곱일 때, 정답 (G×p, G×q) 말고도 **(G, G×p×q)도 최대공약수 G·최소공배수 L 조건을 만족**합니다(엄밀히 답이 유일하지 않음). 교과서 관례는 (G×p, G×q)를 답으로 보지만, 이 관례를 그대로 쓸지 / 문제에 "두 수가 모두 두 자리 수" 같은 제약을 넣어 유일하게 만들지 결정 필요. (원하시면 제약을 코드에 추가하겠습니다.)
- [ ] **그림 의존 심화 유형의 텍스트 출제**: ch52-congtriangle(합동 삼각형 이어 붙인 각도), ch52-paperfold(정사각형 종이접기 각도)는 본래 **그림이 있어야** 상황이 분명한 유형입니다. 그림 없이 글로만 제시하므로, 학생이 상황을 정확히 떠올릴 수 있는지 교실에서 확인. (애매하면 이 두 유형을 심화 풀에서 빼는 것도 가능 — 지시 주시면 처리.)
- [ ] **심화 난이도 적정성**: 등차수열에서 처음 X 넘는 항(ch51-arith), 단위분수 망원 소거(ch51-telescope), 정육면체 칠하기 후 한 면만 색칠(ch52-paintcut) 등은 **상당히 높은 수준**입니다. 우리 학급의 상위권에게 맞는지, 너무 어려우면 제외할 유형이 있는지 판단.

## ④ (참고) 제가 5학년 심화 audit에서 이미 고친 것 — 검증 불필요, 기록용

> 아래는 제가 이번에 **수정 완료**한 항목입니다. 선생님이 하실 일은 없고, 무엇을 손봤는지 투명하게 남깁니다.

- ch52-paintcut: 풀이의 `(n−2)²`(거듭제곱=중등 표기) → `(n−2)×(n−2)`로 교체. 동시에 가드를 강화해 `)²` 형태도 위반으로 잡도록 함(이전엔 숫자 뒤 위첨자만 잡았음).
- ch51-age: 풀이의 문자 미지수 `x` → `□`로 교체(문자 미지수는 중등; 4학년 정비 때 정한 원칙과 통일).
- ch51-arith: 풀이의 문자 미지수 `n` → `□`로 교체 + 나눗셈이 떨어지지 않을 때 풀이에 긴 소수(예: 3.5714285714…)가 노출되던 것 제거.
- ch51-ineq, ch52-fracineqnat: 풀이에 긴 소수(부동소수점 찌꺼기)가 노출되던 부분을 정수·분수 비교로 정리.
