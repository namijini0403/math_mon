## 2학년 — ✅ 전 단원 audit 완료 (2-1·2-2 모두 AUDITED)
- [x] `muli-times` (2-1 곱셈): ✅ `÷` 제거 — "a씩 뛰어 세기"로 몇 배 설명.
- [x] 2-1 세 자리 수·도형: ✅ 자릿값 `×100`·`perShape×count` 곱셈기호 제거 → 자릿값 덧셈·반복 덧셈(곱셈은 2-1 마지막 단원).
- [x] `gugu-missing` (2-2 곱셈구구): ✅ `÷` 제거 — "N단을 외워 N×?=결과" 식으로.
- [x] `len22-conv` (2-2 길이 재기): ✅ `÷100`·`×100` 제거 → "100 cm가 1 m" 묶음 말로.
- [x] `time2-conv`, `time2-units` (2-2 시각과 시간): ✅ `÷60`·`÷24`·`×60`·`×24` 제거 → "60분이 1시간"·"24시간이 하루" 묶음 말로. `time2-read`의 `11×5`(구구 범위 밖)도 "5분씩 N칸"으로.
- [x] `rule2-shape`, `rule2-word` (2-2 규칙 찾기): ✅ `÷`+몫/나머지(3학년) 제거 → "반복되는 N개 중 K번째와 같다"로.
- [x] `rule2-stack` (2-2 규칙 찾기): ✅ `2²·3²`(거듭제곱=전 학년 금지) 제거 → "1×1, 2×2, 3×3…"로.

> 2학년 곱셈(×) 사용 원칙: 곱셈구구(9×9) 범위 안, 곱셈표·곱셈 단원에서만. 자릿값·단위 변환은 덧셈/묶음 말(곱셈 미도입 단원 배려 + 구구 범위 밖 곱 회피). 상세 docs/curriculum/g2.md.

## 3학년 — ✅ 전 단원 audit 완료 (3-1·3-2 모두 AUDITED)
- 3학년부터 **나눗셈(÷)·분수·소수가 정식 도입** → 가드 자동 허용. 표기 위반은 없고, audit은 **풀이 단계화·품질**과 **학기 내 단원 순서**에 집중(상세 docs/curriculum/g3.md).
- [x] 3-1 ①덧셈과뺄셈(Add3): ✅ 한 줄 풀이("받아올림 N번")→ 일·십·백 자리별 받아올림/내림 단계 풀이(addExplain/subExplain). 어림·□역산·문장제도 이유 보강.
- [x] 3-1 ②평면도형(Plane3): ✅ 둘레 역산 `÷2`·`÷4`는 genuine division이라 **÷ 유지**(가드 3학년 허용). 풀이를 "둘레=가로2+세로2", "한 변의 4배" 의미로 단계화. 각·직각 개수에 이유 추가.
- [x] 3-1 ③나눗셈(Div3): ✅ 곱셈↔나눗셈 역산 이유·등분제/포함제 의미를 풀이에 명시.
- [x] 3-1 ④곱셈(Mul31): ✅ 부분곱(일의 자리·십의 자리 따로 곱해 더하기, mul2x1Explain)으로 단계화. 어림·문장제 맥락 보강.
- [x] 3-1 ⑤길이와시간(Time3): ✅ 시간 덧셈/뺄셈을 "분끼리·시끼리"+받아올림/내림 단계로. 단위 변환 ÷·×는 ⑤단원이라 허용·적절.
- [x] 3-1 ⑥분수와소수(Frac3): ✅ 분수=전체 d등분 중 n개, 소수=0.1이 n개 의미로 보강. 비교는 말+기호(앱 결정).
- [x] 3-2 ①곱셈(Mul2): ✅ (세 자리)×(한 자리) 부분곱(mul3x1Explain)으로 단계화. (몇십)×(몇십)·(두 자리)×(두 자리)·문장제 설명 보강.
- [x] 3-2 ②나눗셈(Div2): ✅ (몇십)÷(몇) 자릿값, 나머지 찾는 과정(N단 가장 큰 곱+남은 수)·검산, 문장제 남음 의미 보강.
- [x] 3-2 ③원(Circle): ✅ 지름=반지름×2 관계 단계 풀이(이미 양호, 문장제 완결문장으로).
- [x] 3-2 ④분수(Frac2): ✅ **대분수→가분수 풀이가 비어 있던 것 수정**(자연수×분모+분자 계산 명시). 가→대 ÷나머지, 전체의 분수만큼 보강.
- [x] 3-2 ⑤들이와무게(Measure): ✅ 들이/무게 덧뺄을 작은단위끼리·큰단위끼리+받아올림/내림 단계로(compoundAddSubExplain). 변환·문장제 보강.
- [x] 3-2 ⑥자료의정리(Data): ✅ **표 차 풀이의 절댓값 기호 |…|(중등 표기) 제거** → "큰 수-작은 수" 말로. 그림그래프 역산·문장제 보강.
