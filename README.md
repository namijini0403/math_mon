# 매스몬 (math_mon) 🦊

초등 5학년 수학을 듀오링고처럼 — 문제를 풀어 몬스터를 물리치고, 레벨업해서 타로풍 인증카드를 모으는 학습 PWA.

**2022 개정교육과정 5학년 1·2학기 전체 12개 단원, 83개 문제 유형** 수록:
- 1학기: 자연수의 혼합 계산 · 약수와 배수 · 규칙과 대응 · 약분과 통분 · 분수의 덧셈과 뺄셈 · 다각형의 둘레와 넓이
- 2학기: 수의 범위와 어림하기 · 분수의 곱셈 · 합동과 대칭 · 소수의 곱셈 · 직육면체 · 평균과 가능성

## 특징

- **문제 생성기 (런타임 AI 비용 0원)**: 41개 유형(스킬)을 코드 템플릿 + 시드 RNG로 무한 생성.
  정답은 코드가 계산하므로 항상 정확하고, 오답 보기는 실제 오개념(분모끼리 더하기, ÷2 빠뜨리기 등) 기반.
  스킬당 1,000샘플 속성 테스트로 검증 (총 135개 테스트).
- **난이도 시스템**: 레슨 안에서 기초→보통→도전으로 램프. 기초 정답률이 낮으면 기초에 머물고,
  최근 틀린 유형은 25% 확률로 다시 출제(복습 배지 📌).
- **보스전**: 문제가 제한시간과 함께 떨어진다. 시간 초과·오답이면 블록이 쌓이고 3개면 패배.
  쉬운 문제일수록 시간이 짧고, 보스 HP가 깎일수록 어려워진다. 격파하면 보스 카드 획득.
- **인증카드**: 레벨업 카드 8티어 + 보스 카드 6종, 타로풍. PNG로 저장해서 프사로 사용 가능.
- **연습 모드**: 기초 연산 / 문장제 무한 연습 (하트 없음, 문제당 +1 XP).
- **미션·스트릭·하트·콤보**: 듀오링고식 동기부여 루프. 콤보 5/10/20 도달 시 보너스 XP.
- **출석 보상**: 매일 첫 접속에 보물상자 — 연속 출석일수록 커지는 XP (7일 주기).
- **배지 18종**: 연속 정답·출석·보스 격파·수집 업적을 브론즈/실버/골드 배지로 (프로필에서 컬렉션).
- **교사 현황판**: 반 코드로 학생 진도·정답률·카드 수 확인 (`/#/teacher`).
- **PWA**: 홈 화면 설치, 오프라인 동작 (진도는 로컬 저장 + 서버 동기화).

## 구조

```
client/   React 19 + Vite + Tailwind 4 + framer-motion PWA
  src/generator/   ★ 문제 생성기 (순수 TS — 시드 RNG, 분수 유틸, 오개념 distractor)
  src/game/        스테이지·난이도·XP·카드·저장소(zustand persist)
  src/pages/       유닛맵 / 레슨·보스전 / 연습 / 프로필 / 교사
server/   Hono + Postgres (반 코드 로그인, 진도 동기화, 교사 API)
assets/   ASSET_SPEC.md — 이미지 제작 명세 (이미지 넣기 전엔 플레이스홀더로 동작)
```

## 개발

```bash
npm install
npm test                # 생성기 속성 테스트 (135개)
npm run preview:problems -- u1-simplify 5   # 문제 샘플 미리보기
npm run dev             # 클라이언트 (http://localhost:5173)
npm run dev:server      # 서버 (http://localhost:3000, DATABASE_URL 없으면 DB 기능만 꺼짐)
```

## Railway 배포

1. 이 저장소를 Railway 프로젝트에 연결 (빌드: `npm run build`, 시작: `npm start` — 자동 감지됨)
2. Postgres 플러그인 추가 → `DATABASE_URL` 자동 주입 (테이블은 서버가 부팅 시 자동 생성)
3. 환경변수 `TEACHER_KEY` 설정 (교사 현황판 비밀키)
4. 배포 후 `/#/teacher` 에서 반 코드 생성 → 학생들은 닉네임 + 반 코드 + PIN 4자리로 접속
